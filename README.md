# agentic-coding-playground

A sandbox for exploring, inventing, and iterating on **skills**, **agents**, and **plugins** for coding agents.

Written primarily for [OpenCode](https://opencode.ai) but uses portable formats that should also work with Claude Code, Codex, and other agentic coding tools.

## Structure

- `skills/` — Published skills. Each skill lives in its own subdirectory.
  - `skills/<name>/skills.md` — Full explanation, structure, and workflow for that skill.
- `agents/` — Published agents. Each agent lives in its own subdirectory.
  - `agents/<name>/agents.md` — Full explanation, structure, and workflow for that agent.
- `plugins/` — Published plugins. Each plugin lives in its own subdirectory.
  - `plugins/<name>/plugins.md` — Full explanation, structure, and workflow for that plugin.

## Workflow

Skills, agents, and plugins are invented here and then, when stable, published to the appropriate folder.
